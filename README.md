# Sistema-de-Oculta-o-de-arquivos-em-Imagens-via-LSB
Sistema de esteganografia digital baseado na técnica LSB (Least Significant Bit) para ocultação e extração segura de ficheiros em imagens PNG, com compressão zlib, criptografia XOR e verificação de integridade. Implementado em Python puro com frontend independente.
